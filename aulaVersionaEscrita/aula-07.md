aprendemos sobre funções globais em JavaScript, com foco em setTimeout e setInterval.

setTimeout permite executar um bloco de código uma vez após um período determinado, útil para tarefas como limpar notificações.

setInterval executa um bloco de código repetidamente em intervalos regulares, ideal para criar contadores ou cronômetros.

Para controlar o setInterval, usamos clearInterval para interromper a execução, passando o ID retornado pelo setInterval.

Além disso, aprendemos a importância de consultar a documentação do Mozilla MDN para entender e utilizar corretamente as funções globais do JavaScript. Essas funções são nativas da linguagem e estão prontas para uso, sem necessidade de declaração prévia.
